## Description

The game features a fox in a forest setting in which the player moves the fox around to go over obstacles. The game has a debug screen for users to interact with that is there as a toggle screen.

---

## Setup and Interaction Instructions

Move the fox around the screen using the arrow keys and the space bar to jump. Press '~' to toggle the dubug screen, 'm' for moon gravity mode, and 'h' for the hittest box.
